# Body & Mind

> **Найдите баланс между телом и умом — начните с первого вдоха.**

Лендинг студии йоги, пилатеса и медитации **Body & Mind**. Минималистичный, тёплый и спокойный одностраничный сайт, который рассказывает о пространстве для заботы о себе и приглашает записаться на пробное занятие.

---

## Что внутри

- **Полноэкранный hero** с атмосферным фото и призывом к действию.
- **Раздел «О нас»** — описание философии студии и подхода к занятиям.
- **Три причины начать** — снятие напряжения, улучшение осанки, восстановление энергии.
- **Призыв к действию (CTA)** — запись на пробное занятие и первичную консультацию.
- **Контакты** в футере.

## Технологии

- HTML5
- CSS3 (кастомные свойства, CSS-анимации, адаптивная вёрстка)
- Google Fonts: Cormorant Garamond + Outfit


## Для кого этот сайт

Подойдёт студиям йоги, пилатеса, wellness-пространствам и частным инструкторам, которым нужен быстрый старт: элегантный лендинг с душой, без сложных зависимостей и сборок.

## Как использовать

1. Откройте `index.html` в браузере — сайт работает без сборки.
2. Замените изображения `yoga.jpg` и `yoga2.jpg` на свои.
3. Обновите контакты в футере.
4. Загрузите на любой статический хостинг (GitHub Pages, Netlify, Vercel и т.д.).
